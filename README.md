Proyecto Java hecho con Micronaut

# Generador de informes

## Ejecutar:
```bash
./mvnw mn:run
```

**Quitar banner**
```java

 public static void main(String[] args) {
    Micronaut.build(args)
                 .banner(false)
                 .start();
 }

```
